It was made for fun and to test my knowledge of Vue.

## Getting Started
1. Clone the repository
  ```sh
  git clone https://github.com/LeandroTakara/tic-tac-toe-variants.git
  ```

2. Install the dependencies
  ```sh
  npm install
  ```

3. Run the project
  ```sh
  npm run dev
  ```
